- return の後に空行を入れた方が読みやすいか。step2.でRubyのスタイルガイドや、他の方のコードなどを参考にする。

# 問題文(翻訳)
- 連結リストの先頭ノードheadが与えられたとき、その連結リストに循環構造（サイクル）が存在するかどうかを判定せよ。
- 連結リストにサイクルがあるとは、nextポインタをたどり続けたときに、あるノードに再び到達できるノードが存在する状態のこと。
- 内部部的にはposは「末尾ノードのnextポインタがつながっているノードのインデックス」を表す。ただしposはパラメータとして渡されない。
- 連結リストに循環構造が存在する場合、trueを返す。それ以外の場合はfalseを返す。
